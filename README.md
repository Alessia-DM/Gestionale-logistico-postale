# Gestionale-logistico-postale
Gestionale logistico Full-Stack sviluppato in Spring Boot. Gestione end-to-end di flussi magazzino (DDT, Scatole, Pallet) con architettura MVC e persistenza dati tramite MyBatis Mapper. Include filtri di ricerca avanzati custom, sistemi di etichettatura barcode e sicurezza integrata con Spring Security. Ottimizzato per scalabilità e UX.
